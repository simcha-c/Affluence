# BLISSANA

Blissana is a single-page app, inspired by Asana designed to help teams organize, track and manage their projects through the completion of smaller tasks. 

[Live](https://bliss-ana.herokuapp.com/)

### Technologies Used:
* Ruby
* Ruby on Rails
* PostgreSQL
* React
* Redux
* JavaScript
* HTML5
* CSS
* Heroku
* Webpack

### Features:

* User Authentication
* Team CRUD
* Project CRUD
* Column CRUD
* Task CRUD
